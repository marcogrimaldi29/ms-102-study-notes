MS-102 validates your ability to deploy and manage a Microsoft 365 tenant, implement identity and access with Microsoft Entra, secure the estate with Microsoft Defender XDR, and govern data with Microsoft Purview. It is the integrating exam that sits across every Microsoft 365 workload.

How to use these notes

Work through the four domains in order, then drill the Exam Tips & Caveats page before test day. These notes are a companion — pair every section with hands-on practice in a Microsoft 365 trial tenant and the official Microsoft Learn modules.
